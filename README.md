## 📖 Descripción del Repositorio
Este repositorio contiene todas las prácticas, ejercicios y proyectos desarrollados durante la materia de **Programación Orientada a Objetos (POO)**.

El propósito es aprender desde lo más básico hasta conceptos más avanzados de Python, aplicando buenas prácticas de programación y resolviendo problemas de manera estructurada.

---

## 🧠 Temas Aprendidos

Durante el curso se están abordando los siguientes temas:

✔️ Variables y tipos de datos  
✔️ Estructuras de control (if, for, while)  
✔️ Funciones  
✔️ Clases y objetos  
✔️ Herencia  
✔️ Encapsulamiento

✔️ Buenas prácticas en Python
